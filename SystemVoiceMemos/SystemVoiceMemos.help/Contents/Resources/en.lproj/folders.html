Folders

Organize your recordings with custom folders

Understanding Folders

SystemVoiceMemos uses folders to help you organize recordings. Folders work like playlists — recordings can exist in folders while still being stored in the same location on disk.

Default Folder

All new recordings are saved to the default folder (called "All Recordings" or similar) unless you specify otherwise.

Folder Benefits

  • Organize recordings by project, topic, or date
  • Quickly filter the recordings list
  • Keep related recordings together without moving files

Creating Folders

To create a new folder:

  1. Click the + button in the sidebar or choose New Folder from the File menu
  2. Enter a name for the folder
  3. The folder appears in the sidebar
Note: Folders are virtual organizing structures — they don't create separate directories on disk. All recordings remain in the main recordings directory.

Moving Recordings to Folders

To organize recordings into folders:

  • Drag and drop recordings onto a folder in the sidebar
  • Right-click a recording and choose Move to Folder
  • Select multiple recordings (Shift+click or Cmd+click) and move them together

A recording can be in one folder at a time. Moving it to a new folder removes it from its previous folder.

Viewing Folders

The sidebar shows all your folders:

  • All Recordings — Shows every recording in all folders
  • Folder names — Click any folder to see only its recordings

To toggle the sidebar visibility:

  • Press Cmd+Option+S
  • Choose Toggle Sidebar from the View menu

Renaming Folders

To rename a folder:

  1. Right-click the folder in the sidebar
  2. Choose Rename
  3. Type the new name and press Return

Renaming a folder doesn't affect the recordings it contains.

Deleting Folders

To delete a folder:

  1. Right-click the folder in the sidebar
  2. Choose Delete Folder
  3. Confirm the deletion
Important: Deleting a folder removes the folder itself but keeps all recordings. The recordings that were in the folder will still appear in "All Recordings."

Folder Organization Tips

By Project

Create folders for different projects:

  • Podcast Episodes
  • Music Ideas
  • Lecture Notes
  • Meeting Records

By Date

Organize by time periods:

  • March 2025
  • Q1 Recordings
  • This Week

By Source

Group by what you're recording:

  • Spotify Captures
  • YouTube Audio
  • System Sounds
