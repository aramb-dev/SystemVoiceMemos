Recording

Capture system audio with multi-track support and stem export

Starting a Recording

There are several ways to start recording:

  • Click New Recording in the File menu
  • Press Cmd+N
  • Click the record button in the main window (if available)

Once recording starts, the menu item changes to Stop Recording and a recording indicator appears.

Stopping a Recording

  • Click Stop Recording in the File menu
  • Press Cmd+N again (the shortcut toggles recording)
  • Click the stop button in the main window (if available)

When you stop recording, the audio is automatically saved as an M4A file in your recordings directory.

Multi-Track Recording

SystemVoiceMemos supports recording multiple audio sources simultaneously:

System Audio

Captures any audio playing on your Mac — music from Spotify/Apple Music, video audio, system sounds, etc. This requires Screen Recording permission.

Microphone

Optionally record your voice or external audio devices alongside system audio. This requires Microphone permission and must be enabled in settings.

Note: When multi-track recording is enabled, SystemVoiceMemos creates separate audio tracks for system audio and microphone input. This allows for flexible post-production and stem export.

Stem Export

After recording with multiple tracks, you can export individual stems:

  • Select a multi-track recording
  • Choose Export Stems from the File menu or context menu
  • Choose a destination folder
  • Individual audio files are exported for each track

Stem File Names

Exported stems are named with the recording title plus the track name:

  • RecordingName-system.m4a — System audio track
  • RecordingName-mic.m4a — Microphone track
Use Case: Stem export is useful for podcasters, musicians, and content creators who want to process individual audio tracks separately in a DAW or audio editor.

Recording Names

SystemVoiceMemos automatically names your recordings:

Default Naming

By default, recordings are named with a timestamp format like Recording 2025-03-18 at 14.30.45.

Location-Based Naming

When enabled in Settings, recordings are named with your current city:

  • San Francisco - 2025-03-18 at 14.30
  • New York - Morning Jam

This requires Location permission and can be enabled in Settings.

Custom Names

You can rename any recording by:

  1. Select the recording in the list
  2. Press Return or right-click and choose Rename
  3. Type the new name and press Return

Recording Formats

SystemVoiceMemos saves recordings as M4A files using the AAC codec:

  • Container: M4A (MPEG-4 Audio)
  • Codec: AAC (Advanced Audio Coding)
  • Quality: High (optimized for voice and music)

M4A files are widely compatible and can be played on most devices and software including QuickTime, VLC, iTunes/Apple Music, and more.

Troubleshooting Recording Issues

"Cannot start recording" error

  • Ensure Screen Recording permission is granted in System Settings → Privacy & Security
  • Quit other apps that may be capturing system audio (screen recorders, audio routers)
  • Restart SystemVoiceMemos and try again

Recording has no audio

  • Verify audio is actually playing on your Mac (check system volume)
  • Ensure the correct display is selected for capture in Settings (if using multiple displays)
  • Check that no other audio device is routing sound away from the built-in output

Recording is distorted

  • Lower your Mac's system volume before recording
  • Check if any audio effects or equalizers are enabled in System Settings

See Troubleshooting for more help.
