Settings

Configure preferences, updates, and app behavior

Opening Settings

Open the Settings window:

  • Press Cmd+,
  • Choose Settings… from the app menu (SystemVoiceMemos → Settings)

General Settings

Location-Based Naming

When enabled, SystemVoiceMemos names recordings using your current city instead of timestamps.

  • Enable toggle: Turn on location-based naming
  • Permission: You'll be prompted to grant Location permission
  • Privacy: Location data is only used for naming — never stored or transmitted

Example names with location-based naming:

  • San Francisco - 2025-03-18 at 14.30
  • London Recording
  • Tokyo - Jam Session

Multi-Track Recording

Configure how multiple audio sources are captured:

  • Record System Audio: Always enabled (required for app functionality)
  • Record Microphone: Optionally capture mic alongside system audio
Note: When microphone recording is enabled, you'll need Microphone permission. System audio and microphone are saved as separate tracks for stem export.

Default Output Folder

View or change where recordings are stored:

  • Shows the current recordings directory path
  • Click Reveal in Finder to open the folder
  • Note: Changing this setting may require restarting the app

Updates

Automatic Updates

SystemVoiceMemos uses Sparkle for automatic updates:

  • Checks for updates automatically every 24 hours
  • Notifies you when a new version is available
  • Downloads and installs updates with one click

Check for Updates Manually

To check for updates immediately:

  • Choose Check for Updates… from the app menu
  • Or click Check Now in the Updates section of Settings

Update Settings

Configure update behavior:

  • Automatic checks: Toggle on/off
  • Check interval: How often to check (default: daily)
  • Beta updates: Optionally receive pre-release versions
Note: Updates are downloaded from the official GitHub repository. SystemVoiceMemos is signed and notarized for secure installation.

Appearance

Theme

SystemVoiceMemos follows your macOS system appearance:

  • Light Mode: Uses light color scheme when macOS is in light mode
  • Dark Mode: Uses dark color scheme when macOS is in dark mode
  • Auto: Switches automatically based on time of day

The app uses glass-morphism design elements that adapt to your system theme.

Sidebar

Customize sidebar behavior:

  • Show sidebar on launch: Always show sidebar when app opens
  • Remember sidebar state: Restore sidebar to its last state

Recording Settings

Audio Quality

Configure recording quality:

  • High (default): Best quality, larger file sizes
  • Medium: Good quality, smaller file sizes
  • Low: Smallest files, reduced quality
Note: Higher quality settings create larger files but preserve more audio detail. Most users should keep the default High setting.

Display Selection

If you have multiple displays, choose which display's audio to capture:

  • Primary Display: Captures audio from the main display
  • All Displays: Captures mixed audio from all displays (if supported)

Keyboard Shortcuts

Cmd+,
Open Settings

Resetting Settings

If you're experiencing issues or want to start fresh:

  1. Quit SystemVoiceMemos
  2. Open System Settings → Privacy & Security → Location → SystemVoiceMemos
  3. Disable and re-enable location access if needed
  4. For a full reset, delete ~/Library/Containers/com.arambdev.SystemVoiceMemos/ and relaunch
Warning: Deleting the app container will remove all settings and your recordings. Only do this if you want a completely fresh start.
